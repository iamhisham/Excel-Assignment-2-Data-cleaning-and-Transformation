# Excel Assignment 2 – Data Cleaning and Transformation

## Course
Data Analytics (DA)

## Module
Module 1 – Excel

## Assignment Objective
This assignment focuses on cleaning and transforming a product dataset using Microsoft Excel. The dataset was prepared for analysis by handling missing values, correcting inconsistencies, removing duplicates, restructuring columns, and applying formatting rules.

## Files Included
- Excel_Assignment_2.xlsx
- Project_Documentation.pdf

## Tasks Completed
1. Handling Missing Values
   - Replaced missing Price values using Average (Mean).
   - Filled missing Category values using similar product information.

2. Correcting Inconsistent Data
   - Standardized Product Name formatting.
   - Corrected Category spelling errors.

3. Removing Duplicates
   - Identified and removed duplicate records.

4. Splitting and Merging Data
   - Split Product ID into Manufacturing Date and Country Code.
   - Merged Product Name and Brand Name into Product Brand.

5. Number Formatting
   - Applied currency formatting to Price values.
   - Formatted Manufacturing Date values.

6. Conditional Formatting
   - Applied formatting to highlight Price variations.
   - Highlighted Electronics category records.
